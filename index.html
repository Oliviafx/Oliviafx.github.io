Projects

Simulating Evolving Artificial Life | September-December 2016

This is a project done as part of the final project of CS 2112: Object Oriented Programming and Data Structures Honors at Cornell University.

In this project, a critter world was created in which each critter had their own language and program to execute. The project included having the parse the programs and create Abstract Syntax Trees from the programs, interpreting the programs, designing a GUI in order to show the world of the critters, and making the project distributed by creating a thread safe server and client.

Pawpulation | Animal Health Hackathon 2017

Created a web application in Java with a server and a client that would allow vets to access a database of epidemiological data, letting the user to upload, search, and download the data in CSV format. Used Postgres to implement the database.

Recommendations | March-April 2017

Created a website that could get food recommendations from Yelp's Fusion API based off location and price willing to pay by the user. Used Google Maps API to show the exact location of the place in order to help aid the user. Used HTML, CSS, Javascript, and EJS for front end, and used NodeJS to create a server for the backend.

Text Editor | September 2016

Implemented a hash table, a trie and a bloom filter from scratch in Java. Used these data structures to give a text editor GUI working functionality, including autocompletion, search, and spell check.

Encryption | September 2016

Implemented the RSA algorithm, as well as the Caesar and Vigenere encryption ciphers in Java using the Factory Design Pattern. Worked with input/output streams to encrypt and decrypt large file streams.
